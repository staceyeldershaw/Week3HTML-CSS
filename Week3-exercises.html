CSS Font Size Units

px stands for pixel. This div uses px (pixels): 24px - absolute unit which means the units are fixed and the length or size expressed will appear exactly as specified, 1px=1/96th of 1inch
pt stands for point. This div uses pt (points): 10pt - absolute unit, 1pt = 1/72nd of 1inch, this is another absolute unit. Traditionally it is used in print media and typography. On screens, browers automatically convert points to pixels, which does vary depending on screen resolution and settings, but 1pt ~ 1.333px.
This div uses em: 1.5em - relative to parent font size, so if the parent element has a font size of 16px, then 1.5em would be 24px. If the parent element's font size changes, the size of this div will also change accordingly.Relative length units specify a length relative to another length property. Relative length units scale better between different rendering medium.
This div uses rem: 2rem - relative to root font size, so if the root element (html) has a font size of 16px, then 2rem would be 32px. Similar to em, but it always refers to the root element's font size, regardless of nesting.

Overflow

Overflow is a CSS property that controls what happens when content exceeds the dimensions of its container. It can be set to values like 'visible', 'hidden', 'scroll', or 'auto' to determine how excess content is handled. Everything in CSS is a box. You can constrain the size of these boxes by assigning values such as width and height. Overflow happens when there is too much content to fit in a box. CSS provides various tools to manage overflow.
